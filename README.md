# Hisab DPUA

Aplikasi web **Hisab Awal Bulan Kamariah** — menghitung dan menyajikan laporan visibilitas hilal untuk penetapan awal bulan Hijriah di berbagai kota.

### 🔗 [Buka aplikasi → hisab-dpua.github.io/hisabdpua-frontend](https://hisab-dpua.github.io/hisabdpua-frontend/)

## Metode hisab

| Metode                 | Keterangan                        |
| ---------------------- | --------------------------------- |
| **Irsyadul Murid**     | Hisab kontemporer (high accuracy) |
| **Ad-Durr Al-Anīq**    | Hisab kontemporer (high accuracy) |
| **Maghīb al-Qamarain** | Marja' regional berbasis bujur    |

## Fitur

- **Laporan multi-kota** untuk satu metode dalam satu tabel
- **Laporan Maghīb al-Qamarain** (tabel harfiah & koreksi kontinuitas)
- **Panel admin**: persetujuan pengguna baru + kelola data kota

## Cara akses

Daftar akun → menunggu persetujuan admin → setelah disetujui, fitur hisab terbuka.
